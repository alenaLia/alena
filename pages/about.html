About OpenSeat

A simple, student-powered tool to find the best study spot in the Stevens library.

What is OpenSeat?

OpenSeat is a crowdsourced platform that helps Stevens students find available study spaces in the library. By reporting crowd levels on each floor, students help each other make informed decisions about where to study. Built with simple web technologies and hosted on GitHub Pages, OpenSeat (in-theory) provides real-time updates.

Why We Built This

Finding a quiet study spot during exam season can be frustrating. We built OpenSeat to save students time and reduce the stress of wandering between floors looking for an open seat. Our mission is to make studying more efficient and accessible for the entire Stevens community. Also we had to if we wanted to pass CS-146. But we digress.
