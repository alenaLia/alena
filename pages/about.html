About OpenSeat

A simple, student-powered tool to find the best study spot in the Stevens library.

What is OpenSeat?

OpenSeat is a crowdsourced platform that helps Stevens students find available study spaces in the library. By reporting crowd levels on each floor, students help each other make informed decisions about where to study.

How It Works

  1. Students Report: When you're in the library, take a moment to report how crowded your floor is using the quick form on the homepage or by scanning a QR code.
  2. Data Updates: Your report is combined with others to show real-time crowd averages for each floor.
  3. Find Your Spot: Check the homepage before heading to the library to see which floors have space available.

Why We Built This

Finding a quiet study spot during exam season can be frustrating. We built OpenSeat to save students time and reduce the stress of wandering between floors looking for an open seat.

Technology

OpenSeat is built with simple, accessible web technologies:

  • Plain HTML, CSS, and vanilla JavaScript
  • No frameworks or build tools required
  • Hosted on GitHub Pages for free, reliable hosting
  • Mobile-first, responsive design
  • Accessible to all students, including those using assistive technologies

Privacy & Data

We respect your privacy. OpenSeat collects only the floor number and crowd level you report. We don't collect names, student IDs, or any personal information. All data is anonymous and used solely to display aggregate crowd levels.

Get Involved

OpenSeat is an open-source student project. If you'd like to contribute ideas, report bugs, or help improve the platform, visit our GitHub repository or contact the team.
